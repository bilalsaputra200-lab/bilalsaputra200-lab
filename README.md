### 🧑‍💻 Tentang Saya

Saya adalah seorang profesional IT (S.Kom) dari STMIK IKMI Cirebon yang berfokus pada pengembangan web dan infrastruktur *backend*. Saya memiliki *passion* dalam merancang arsitektur sistem berskala besar yang efisien dan berdampak nyata.

- 🏥 **Fokus Saat Ini:** Pengembangan dan integrasi Sistem Informasi Manajemen Rumah Sakit (SIMRS), termasuk eksplorasi *open-source* seperti SIMRS Khanza.
- ⚡ **Eksplorasi Teknologi:** Membangun *backend* berkinerja tinggi menggunakan **Golang** untuk skalabilitas tingkat lanjut, bersanding dengan ekosistem **PHP**.
- 💼 **Inisiatif:** Sedang mengembangkan dan membangun ekosistem perangkat lunak di bawah naungan **B.S. Software**.
- 📫 **Tanya saya tentang:** *Web Development, Database Management, IT Support, System Architecture.*
